# 50. Pow(x, n)

#### [link](https://leetcode.com/problems/powx-n/) 
Implement [pow(x, n)](http://www.cplusplus.com/reference/valarray/pow/), which calculates x raised to the power n (x^n).

#### Description

#### Example 1:
```
Input: 2.00000, 10
Output: 1024.00000
```
#### Example 2:
```
Input: 2.10000, 3
Output: 9.26100
```
#### Example 3:
```
Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
```

#### Note:
* -100.0 < x < 100.0
* n is a 32-bit signed integer, within the range [−2^31, 2^31 − 1]

## Fast Power
```java
class Solution {
    public double myPow(double x, int m) {
        long n = (long) m;
        if(n < 0){
            n *= -1;
            x = 1 / x;
        }
        double ans = 1.0;
        while(n != 0){
            if(n % 2 == 1)
                ans *= x;
            n /= 2;
            x *= x;
        }
        return ans;
    }
}
```

### Time complexity
* O(log(n))
### Space complexity
* O(1)
