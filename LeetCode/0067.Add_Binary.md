# 67. Add Binary

#### [link](https://leetcode.com/problems/add-binary/)

#### Description
Given two binary strings `a` and `b, return *their sum as a binary string*.

#### Example 1:
```
Input: a = "11", b = "1"
Output: "100"
```
#### Example 2:
```
Input: a = "1010", b = "1011"
Output: "10101"
```

#### Constraints:
* `1 <= a.length, b.length <= 10^4`
* `a` and `b` consist only of `'0'` or `'1'` characters.
* Each string does not contain leading zeros except for the zero itself.

## Bit Manipulation
```java
class Solution {
    public String addBinary(String a, String b) {
        int x = a.length() - 1;
        int y = b.length() - 1;
        int carry = 0;
        StringBuilder ret = new StringBuilder();
        
        while(x >= 0 || y >= 0){
            if(x >= 0)
                carry += a.charAt(x--) - '0';     
            if(y >= 0)
                carry += b.charAt(y--) - '0';
            ret.append(carry % 2);
            carry = carry / 2;
        }
        if(carry > 0)
            ret.append(carry);
        
        return ret.reverse().toString();
    }
}
```
### Time complexity
* O(n)
### Space complexity
* O(n)

## Bit Manipulation (python)
```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        i = len(a) - 1
        j = len(b) - 1

        sum_chars = []
        carry = 0
        while i >= 0 or j >= 0:
            if i >= 0:
                carry += int(a[i])
            if j >= 0:
                carry += int(b[j])
            sum_chars.append(str(carry % 2))
            carry //= 2

            i -= 1
            j -= 1

        if carry == 1:
            sum_chars.append(str(carry))

        return "".join(reversed(sum_chars))
```
### Time complexity
* O(max(m, n))
### Space complexity
* O(max(m, n))

## XOR + Shift
```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        x, y = int(a, 2), int(b, 2)
        while y != 0:
            x, y = (x ^ y), (x & y) << 1
        
        return bin(x)[2:]
```
### Time complexity
* O(max(m, n))
### Space complexity
* O(1)
